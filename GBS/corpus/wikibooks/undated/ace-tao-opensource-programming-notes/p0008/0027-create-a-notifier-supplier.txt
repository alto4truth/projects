TAO's Notification service implements the OMG's specification for an extension to the event service. This new specification adds QOS and event filtering to the existing event model. For most applications, this infrastructure is a bit beyond what's needed, and for that reason, there aren't examples included in this book. If you wish to contribute a model, feel free.
