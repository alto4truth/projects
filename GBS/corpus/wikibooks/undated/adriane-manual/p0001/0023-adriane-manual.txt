cd

umount /media/sdb1

There is a special aspect in this example: the name of the music folder contains a blank space. To let the cp command know that not two different folders are meant, but that it is the folder’s name with a blank space, the folder’s name it set into “ “ (quotation marks). Otherwise the cp command understands that "My Music" are two different folders (My and Music).
After the copy operation completes the media player (menu "Multimedia") is ablte to open the directory, where the files were copied to, using "Browse files".
By the way, the mount or unmount command in this example is only required, if the USB stick was not previously mounted by the file manager.
The shell can be exited using the exit command.
