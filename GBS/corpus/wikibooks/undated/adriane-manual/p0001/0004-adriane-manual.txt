= Working with Adriane =

== Start and help ==
If you are using the Adriane version of Knoppix it starts automatically. Otherwise you will have to type "Adriane" when the boot screen is shown.
First you will hear the speech output reporting the first menu item "Help with enter key, arrow key down for next menu". By pressing enter you will reach a help window, which explains the basic language output functions. Here you can also select individual functions and components. With the ARROW down key you can reach the other programs and step through the menu.
