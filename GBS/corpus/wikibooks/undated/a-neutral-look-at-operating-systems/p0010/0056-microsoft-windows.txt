== ReactOS ==
ReactOS is a project attempting to create a Windows compatible open-source operating system. It uses no code from Microsoft Windows and is completely open-source. It also is not based on Linux, however some code is carried over from the Wine project which runs on Linux. Currently it is in alpha stage and not suitable for production systems.
