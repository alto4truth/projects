function_name identifies which function it is you want to use (you'd figure...). For example, the function raw_input(this function has been renamed to input in Python 3), which will be the first function that we will use.
Parameters are the values you pass to the function to tell it what it should do, and how to do it.. for example, if a function multiplies any given number by five, the stuff in parameters tells the function which number it should multiply by five. Put the number 70 into parameters, and the function will do 70×5.
