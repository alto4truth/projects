So what happened?
when 'def hello()' was run, a function called 'hello' was created
When the line 'hello()' was run, the function 'hello' was executed (The code inside it was run)
The function 'hello' printed "hello" onscreen, then returned the number '1234' back to the main program
The main program now sees the line as 'print 1234' and as a result, printed '1234'
That accounts for everything that happened. remember, that the main program had NO IDEA that the words "hello" were printed onscreen. All it saw was '1234', and printed that onscreen.
