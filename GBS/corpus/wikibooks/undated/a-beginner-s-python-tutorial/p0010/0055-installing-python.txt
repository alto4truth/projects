==== Mac users ====
Starting from Mac OS X (Tiger) and ending with MacOS 12.3, Python shipped by default with MacOS. That said you might want to update to the newer version (check the version and install status by starting python in a command line terminal). Also IDLE (the Python editor) might be missing in the standard installation. If you want to (re-)install Python, you can get it from the Mac download page on the Python site.
