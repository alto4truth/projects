== Problem presentation ==
Each step of the Learning Track will have links to problems in order from simple to complex. This will often not be in numerical order. A link is provided to each problem page which has:

The problem statement: Describes problem, and ask you to implement a solution.
An example: Presents a example of the function with input and the correct result.
A unit test: A complete program that executes multiple tests of problem solution. You can use this program on the Elm online editor, http://elm-lang.org/try. The stubbed-out function you need to implement will be at the top, just below the imports.
Hints: Suggests one or more ways to implement a solution.
Solutions: Links to a page with one or more solutions.
