As the stack grows backwards, the return address is therefore stored as a little-endian number in memory.
PC is set to the target address.
Flags: none

Return from Subroutine: RTS
Return from a subroutine to the point where it called with JSR.
The return address is popped from the stack (low byte first, then high byte).The return address is incremented and stored in PC.
Flags: none

Return from Interrupt: RTI
Return from an interrupt.
SR is popped from the stack.
PC is popped from the stack.
Flags: all
