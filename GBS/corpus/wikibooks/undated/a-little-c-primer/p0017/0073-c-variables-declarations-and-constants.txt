The string "This is a test!" is used to initialize "s" through the
"strcpy()" function, discussed in a later chapter. The stored string will
contain a terminating "null" character (the character with ASCII code 0,
represented by '\0'). The null is used by C functions that manipulate
strings to determine where the end of the string is, and it is important to
remember the null is there.
