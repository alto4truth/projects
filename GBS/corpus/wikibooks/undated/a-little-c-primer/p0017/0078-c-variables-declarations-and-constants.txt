C contains a concept similar to a structure known as a "union". A union is declared in much the same way as a structure. For example:

The difference is that the union can store either of these values, but not
both at the same time. A "char" value or an "int" value can be stored in an
instance of the union defined above, but it's not possible to store both at
the same time. Only enough memory is allocated for the union to store the
value of the biggest declared item in it, and that same memory is used to
store data for all the declared items. Unions are not often used and will
not be discussed further.
