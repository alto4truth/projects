/* frdata.c */

#include <stdio.h>

void main()
{
int n1;
float n2;
FILE *fp;

fp = fopen( "data", "r" );

printf( "%d %f", n1, n2 );
fclose( fp );
}

The "fscanf()" function uses the same format codes as "fprintf()", with the
familiar exceptions:

There is no "%g" format code.
The "%f" and "%e" format codes work the same.
There is a "%h" format code for reading short integers.
Numeric modifiers can be used, of course. The "fscanf()" function returns
the number of items that it successfully read, or the EOF code, an "int", if
it encounters the end of the file or an error.
The following program demonstrates the use of "fprintf()" and "fscanf()":
