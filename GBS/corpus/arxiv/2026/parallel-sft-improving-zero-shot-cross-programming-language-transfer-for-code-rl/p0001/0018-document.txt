These datasets contain mostly solutions in Python. To generate parallel programs in other languages, we perform
synthetic code translation: for each instance (q, cPython), we translate cPython into cC++, cJava, etc., by prompting Llama-4Maverick, a 400B-parameter mixture-of-expert model (Meta AI, 2025). We include our translation prompt in §C. This
model generally has good translation accuracy: it achieves 57.3% pass@1 when validated on the tests in the datasets.
We further filter out all translations that fail the tests to ensure translation quality.7
