.1 Problem Formulation
We formalize a multi-agent system A as being parameterized by a collection of configurable natural
language elements, which we termtextual parameters Θ. These parameters encompass all the system’s textual components, such as system prompts
for each agent {Pi}, descriptions of available tools
{Dj}, and the rules governing inter-agent communication.
Given a downstream task distribution T , represented by a dataset of input-output pairs (Q, A) =
{(qk, ak)}N
k=1, the system A(Θ) takes an input
query qk and produces a final outputˆak. This execution generates a corresponding trajectory τk, which
is a detailed record of the agents’ intermediate reasoning steps, tool calls, and communications.
The objective is to discover the optimal set of
textual parameters Θ∗ that maximizes the system’s
performance as measured by a reward metric R
(e.g., success rate). This optimization problem can
be expressed as:
