C Implementation Details
Models and optimization budget.We implement TPGO through an OpenAI-compatible
API interface. Unless otherwise specified,
gemini-2.5-pro is used for all meta-optimization
components, including theParser LLM, theGradient Generator(Reflector), and theOptimizer
LLM. These modules use temperature 0.7 and topp= 0.95 . Each API call is retried up to three
times with exponential backoff. In all main experiments, we run TPGO for at most five optimization
iterations.

subset reported in the main results, covering146
tasksfrom four domains:Repository Management,3D Designing,Browser Automation, and
Web Searching. This benchmark is used in an
exploratory optimizationsetting, where the Gradient Generator analyzes execution traces without
access to gold answers. ForGAIA(Mialon et al.,
2023), we start from the public validation split of
165 questionsand filter it to atext-onlysubset

22), matching our focus on textual reasoning and
