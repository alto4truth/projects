Graph representation and updates.Each agent
configuration is represented as a hierarchical JSONserialized PromptNode structure corresponding to
the Textual Parameter Graph (TPG). Nodes are
typed as role, logic, or tool; a legacy generic
type is retained for compatibility with existing
prompts. Graph updates are applied atomically
at each iteration by a prompt manager that supports
node rewriting, insertion, and deletion.
Textual gradients and clustering.For each trajectory τk, the Reflector generates negative textual gradients δ− that capture failure patterns and
positive textual gradients δ+ that summarize effective behaviors. Negative gradients are embedded
and stored in a persistent vector database. We remove near-duplicate entries by cosine-similarity
filtering, then cluster the remaining gradients using DBSC
