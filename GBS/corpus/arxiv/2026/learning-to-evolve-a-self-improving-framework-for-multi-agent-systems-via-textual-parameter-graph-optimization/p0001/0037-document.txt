6 Conclusion
We introduced Textual Parameter Graph Optimization (TPGO), a framework that improves agent
systems through structured graph evolution rather
than flat prompt tuning. By decomposing agent
configurations into modular textual components,
refining them with execution-grounded textualgradient signals, and leveraging optimization memory through GRAO, TPGO enables more stable
and effective agent optimization. Experiments on
MCP-Universe and GAIA show consistent gains in
both effectiveness and efficiency, suggesting that
structured, experience-informed optimization is a
promising approach for reducing the manual burden of agent engineering.
