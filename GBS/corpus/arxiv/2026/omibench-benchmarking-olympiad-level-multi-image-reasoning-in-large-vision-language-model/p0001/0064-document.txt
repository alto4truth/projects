A.3. Details of Rationale Annotations
Most existing competition-style datasets provide only final answers or brief solution sketches, which are
insufficient for analysing model reasoning behaviours. To address this limitation, we construct expert-verified,
step-by-step rationales for each problem using a two-stage pipeline that combines LLM generation with
careful human verification.
A.3.1. LLM-Assisted Rationale Generation
We first use Gemini-2.5-pro-thinking to generate multiple candidate solutions for each problem.
