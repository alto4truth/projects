Stage 4: Region description.LetR={r i}, i= 1, ..., N
denote the set of generated region coordinates. In this stage,
we aim for region-level verification questions conditioned
onR. Specifically, we prompt the LVLM to describe each
specific region in the image independently, using the following template: “Describe{coordinate}in the image in
detail.”, where{coordinate}is one of ther i ∈ R. We repeat this step multiple times for each region to produce a set
of diverse answers, which proves to be beneficial for performance improvements, as shown in the experimental section.

the following forms of image prompts: (i) the original image, (ii) the image overlaid with the bounding box coordinatesr i, and (iii) a crop from the original image using the
