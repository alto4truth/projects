3.3 Optional UMLS Alias Mapping
We curated a UMLS-based alias map for all 134 CRF items (coverage
134/134) and integrated it as an optional pipeline mode (--use-umls-map
ping) to reduce key-name brittleness in model outputs. The map associates
each official item name with its UMLS CUI and preferred name, plus manually
verified synonyms. This mode is activated in all final submissions.
