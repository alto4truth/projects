The AVISE framework, illustrated in Figure 1, can
be used to evaluate the security of an AI system
by running automated black-box and grey-box attacks or white-box assessments, or all of the above,
on the system. In this section, we will demonstrate
how the Red Queen SET developed in Section 4
can be used to discover jailbreak vulnerabilities in
language models. The experiments were ran on
Ubuntu 24.04 using two NVIDIA Tesla P100 (16
GB) Graphical Processing Units (GPUs) and 234.4
GB of Random Access Memory (RAM).
