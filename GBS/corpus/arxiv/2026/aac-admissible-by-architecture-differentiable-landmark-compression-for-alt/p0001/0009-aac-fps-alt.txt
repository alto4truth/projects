E.2 Directionality-Aware Accounting Convention
Convention.Matched memory atBbytes per vertex matches the per-vertex deployed-label storage of the
two arms. On directed graphs ALT stores both forward and backward landmark distances (2Kfloats per
vertex), so matched memory atBB/v is AACm=B/4vs. ALTK=B/8. On undirected graphs ALT stores
a single direction (Kfloats per vertex), so matched memory atBB/v is AACm=B/4vs. ALTK=B/4.
