A Ge'ez Script Character Matrix
The Ge'ez script is an abugida in which each base consonant has seven vowel-order variants. Figure 7 shows
the core character matrix used in Tigrinya. The visual similarity between vowel orders of the same consonant
family and between certain distinct consonant families illustrates the fine-grained discrimination required for
accurate recognition.
Figure 7: Tigrinya Ge'ez fidel matrix showing 33 base consonants with 7 vowel orders (231 syllographs), 4
labialized consonant groups (20 forms), and 8 punctuation marks.
