4.2 Policy Learning
After the plans have been generated, we train a
reinforcement learning agent using the stepwise
plan observation setting (Subsection 3). At each
timestep, the agent observes the environment and
receives an embedding of the current plan step. It
must learn to align actions with plan steps based
on a delayed reward signal provided only upon
successful completion of the entire plan. We use
the PPO algorithm to train the policy.
