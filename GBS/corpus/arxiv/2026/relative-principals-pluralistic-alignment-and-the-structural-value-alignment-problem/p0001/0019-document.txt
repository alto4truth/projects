Instead of focusing on the normative component of the value alignment problem (“What are the correct values
to encode in AI systems?”) or the technical component (“How do we encode said values?”), this description
emphasises thecontextsin which misalignment can arise—i.e., the socio-dynamic structure of value alignment.
Within this framework, the value alignment problem for AI can be understood as varying along three orthogonal
but interacting axes:
(1) Theobjectives axis, which describes the extent to which the system’s formal objectives (i.e., objective
functions) accurately capture the principal’s true goals.
(2) Theinformation axis, which describes the degree of transparency, observability, and verifiability between
the system and its human principals.
